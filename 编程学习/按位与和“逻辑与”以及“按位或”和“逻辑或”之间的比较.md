`&` 和 `&&` 的比较：

- `&` ：需要将两边表达式的结果都计算出才能进行判断，而且对两边表达式的结果类型还有要求（都要求结果要么为 `1` 要么为 `0` ）。
-  `&&` ：当表达式 1 的结果为 0 时，即可进行判断，**不会再计算后面表达式的结果**，除此之外还对表达式结果类型没有要求。
- 当用于逻辑判断时，尽量使用 `&&` 。


`|` 和 `||` 的比较：

- `|` ：需要将两边表达式的结果都计算出才能进行判断，而且对两边表达式的结果类型还有要求（都要求结果要么为 `1` 要么为 `0` ）。
-  `||` ：当表达式 1 的结果为 1 时，即可进行判断，**不会再计算后面表达式的结果**，除此之外还对表达式结果类型没有要求。
- 当用于逻辑判断时，尽量使用 `||` 。

	int i = 0, a = 0, b = 2, d = 4;
