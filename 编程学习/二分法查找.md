核心思想：不断缩小目标所在区间，从而最后锁定目标值

请编写一个二分法查找函数：

```c
//二分查找函数
// 函数的参数：参数1 -> 整型数组 参数2 -> 要查找的值
//函数的作用：在一个数组中利用二分法快速查找值，并返回相应的下标

int bin_search(int arr[], int key) {
	int left = 0;
	int right = sizeof(arr) / sizeof(arr[0]) - 1;
	int mid = -1;

	while (left <= right) {     //查找的极限为：right = left = mid
		mid = (right + left) / 2;
		if (arr[mid] < key) {
			left = mid + 1; //这个"+1"不能省略。试想一下，如果将目标值范围缩小为两个值之间，而目标值为两个值中的左侧的那一个，如果把"+1"省略，最后会进入死循环，永远找不到那个目标值。
		}
		else if (arr[mid] > key) {
			right = mid - 1; //这个"-1"可以省略，想想为什么？（提示：这和整数的除法有关，即这条代码：mid = (right + left) / 2;）
		}
		else {
			return mid;//找到，返回下标
		}
	}

	return -1;//没找到，返回-1
}
```

> 对于整数数组，利用二分法查找，在缩小区间时（左值或右值发生更换）需要多注意些。
