全局变量的外部链接属性：如果一个变量是全局变量，则可以跨文件使用该变量，只要在使用之前对该全局变量进行申明。 ^067f87

test.c 文件：
```c
int a = 120; //定义一个全局变量
```
main.c 文件：
```c
#include <stdio.h>

extern int a; //申明该全局变量

int main() {

	printf("%d\n", a); //使用该全局变量

	system("pause");
	return 0;
}
```
