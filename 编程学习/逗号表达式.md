**语法：** `exp1,exp2,exp3,……`

逗号表达式就是逗号隔开的一串表达式

**特点：** 从左向右依次计算，整个表达式的结果是最后一个表达式的结果

```c
int main() {

	int a = 9;
	int b = 10;
	int c = 0;

	//        c = 7     a = 17      14
	int d = (c = a - 2, a = b + c, a - 3); //逗号表达式，从左向右依次计算

	printf("%d\n", d); //结果为14
	system("pause");
	return 0;
}
```
